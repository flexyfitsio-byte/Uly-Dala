# Design tokens — Uly Dala

## Color
- --navy-deep: #23263D   (основной фон, как в лого/прототипе)
- --navy-panel: #2E3152  (карточки, панели, шапка)
- --gold: #C9A15A        (акцент, рамки, иконки — как сабли/юрта в лого)
- --gold-soft: #E4C989   (hover, вторичный акцент)
- --cream: #F3EFE6       (текст на тёмном, светлые поверхности)
- --ink: #1A1C2E         (текст на светлом)
- --muted: #9A9CBE       (второстепенный текст на тёмном)

## Type
- Display (лого, заголовки): условный "стенсил-геометрик" — используем 'Oswald', с большим tracking и uppercase, как в лого ULY DALA
- Body/UI: 'Manrope' — чистый геометричный гротеск
- Числа/акценты (цены, XP): 'Oswald' tabular

## Signature element
Орнаментальная рамка (казахский узор) вокруг форм авторизации и карточек достижений —
повторяет золотую рамку из прототипа логина. Плюс силуэт юрты как разделитель секций.

## Layout
Тёмная навигационная шапка (как фон лого), светлые/золотые акценты на карточках мест,
нижняя навигация адаптирована в верхнее меню для десктопа (карта / гид / профиль / партнёрам).
